Software

Open Source software

All tags Bioinformatics

BASTA

Customisable sequence taxonomy annotation tool. It assigns taxonomies to sequences or groups of sequences based on 7 level NCBI taxonomies and a user defined sequence comparison file. Additionally, BASTA can be used to filter fasta files based on taxonomic classification and visualise results in KRONA plots.

Panbiom

Panbiom is a tool to estimate the core operational taxonomic units (OTUs) of a user-defined set of samples and replicates.

ampli-tools

Collection of helper scripts for amplicon sequence analysis.
